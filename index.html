Mauro's Reflex Arena Utilities

Some basic, but useful, tools for moving stuff around in your Reflex Arena Maps

Brush Shifter Utility

Useful for shifting all brush geometry in your map by a given amount in each axis

Usage

ReflexBrushShifter <myMap>.map shift-X shift-Y shift-Z

Where <myMap>.map is your input map file and shift-X, shift-Y, shift-Z are the values by which you wish to shift every entity.

Entity Shifter Utility

Useful for shifting all point entities in your map by a given amount in each axis

Usage

ReflexEntityShifter <myMap>.map shift-X shift-Y shift-Z

Where <myMap>.map is your input map file and shift-X, shift-Y, shift-Z are the values by which you wish to shift every entity.

Map Center Utility

This tool will shift all brush-based geometry and point entities such that the three-dimensional center of the map will coincide with the three-dimensional origin point of the world

Usage

ReflexMapCenter <myMap>.map

Where <myMap>.map is your input map file to be centered.
